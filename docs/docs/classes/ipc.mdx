---
title: IPC
---

The main hub for recieving messages and interacting with other clusters.

## Constructor

```js
new Client(cid, max);
```

| PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION |
| :-------: | :--: | :------: | :-----: | :---------: |
| cid | [number](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number) |  ❌   | `undefined`  | Cluster ID. |
| max | [number?](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number) |  ✔️   | `50`  | Max number of listeners. |

## Methods

### send
Sends an IPC event to all clusters. This will trigger all
listeners for given event on all clusters with given arguments.
:::note
Any arguments given will become dereferenced so ensure
any references are deep cloned.
:::
```js
send(event, ...args);
```
| PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION |
| :-------: | :--: | :------: | :-----: | :---------: |
| event | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) |  ❌   | `undefined`  | Event name to envoke listeners from. |
| args | any[]? |  ✔️  | `undefined`  | Arguments to pass through CallableFunction. |

#### Returns
void
___

### sendTo
Sends an IPC event to a certain cluster. This will trigger all
listeners for given event on given cluster with given arguments.
:::note
Any arguments given will become dereferenced so ensure
any references are deep cloned.
:::
```js
send(cluster, event, ...args);
```
| PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION |
| :-------: | :--: | :------: | :-----: | :---------: |
| cluster | [number](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number) |  ❌   | `undefined`  | Id of cluster to send to. |
| event | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) |  ❌   | `undefined`  | Event name to envoke listeners from. |
| args | any[]? |  ✔️  | `undefined`  | Arguments to pass through CallableFunction. |

#### Returns
void
___
